API do Livro dos Espíritos

Para receber o JSON com todas as perguntas, utilizar a URL com o /GET na frente, da seguinte forma:

https://livrodosespiritos.herokuapp.com/get

Para receber o JSON com uma pergunta específica, adicionar o número da pergunta após o /GET, da seguinte forma:

https://livrodosespiritos.herokuapp.com/get/123
